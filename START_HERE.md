## 📝 Customization Quick Guide

### Update Your Info
Edit `index.html` - search for text and replace

### Change Colors
Edit `css/style.css`:
```css
#49c5b6  /* Cyan */
#FF9398  /* Pink */
```

### Add Your CV
Place PDF in `assets/docs/Kiarash_Farajzadehahary_CV.pdf`

### Add Works/Publications
Edit `pages/works.html` - copy existing work-card divs

### Add New Page
1. Copy `pages/works.html`
2. Rename it
3. Update navigation in all files
4. Add your content
